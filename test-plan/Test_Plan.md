# Test Plan – SauceDemo Web Application

## 1. Objective
The objective of this testing effort is to validate the core functionality of the SauceDemo web application, ensuring that key features work as expected and defects are identified.

## 2. Scope

### In Scope
- User login functionality
- Product listing page
- Shopping cart
- Checkout process

### Out of Scope
- Performance testing
- Security testing
- Mobile responsiveness (optional)

## 3. Test Approach
Testing will be conducted manually using predefined test cases. Functional and negative testing techniques will be applied.

## 4. Test Environment
- Browser: OperaGX
- Operating System: Windows 11 Home
- Test URL: https://www.saucedemo.com/

## 5. Test Types
- Functional Testing
- UI Testing
- Negative Testing
- Exploratory Testing

## 6. Entry Criteria
- Application is accessible
- Test cases are prepared

## 7. Exit Criteria
- All test cases executed
- Critical defects documented

## 8. Risks & Assumptions
- Limited access to backend systems
- Testing limited to UI behavior

## 9. Deliverables
- Test Plan
- Test Cases
- Bug Reports
- Test Summary Report
